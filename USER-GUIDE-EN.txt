3. HOW SHOULD THE IMAGES BE NAMED?
   ========================================

Number the images according to the order in which you want
them to appear:

1.jpg
2.jpg
3.jpg
4.jpg

You may also use descriptive names:

1 - Introduction.jpg
2 - Characters.jpg
3 - World Map.jpg
4 - Ending.jpg

The application uses the first number found in each file name
to determine the slideshow order.

The safest naming method is:

1, 2, 3, 4, 5...

.png

.jpeg
.webp
.gif

5. HOW TO START THE SLIDESHOW
   ========================================

6. Double-click the .alek file that matches your preferred speed.

7. Click the blue moon on the opening screen.

8. The slideshow will automatically begin with the first image.

9. After the final image, the slideshow will return to the
   first image and continue playing.

10. KEYBOARD CONTROLS
    ========================================

Spacebar:
Pauses or resumes the slideshow and music.

Right Arrow:
Moves to the next image.

Left Arrow:
Returns to the previous image.

F11:
Enters or exits full-screen mode.
